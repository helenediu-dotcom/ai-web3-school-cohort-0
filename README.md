# AI × Web3 School — Cohort 0 学习仓库

> Helene 的 AI × Web3 School 个人学习记录仓库

## 学员信息

- **背景**：环保行业环境咨询，AI / Web3 / 编程三方面新手
- **每日投入**：1-2 小时
- **学习周期**：4 周
- **开始时间**：2026-05

## 仓库结构

```
ai-web3-school-cohort-0/
├── notes/                        # 学习笔记（按周分组）
│   ├── week1-ai-basics/          # Week 1：LLM、Prompt、Context、Agent
│   ├── week2-web3-basics/        # Week 2：Network、Cryptography、Wallet、Smart Contract
│   ├── week3-ai-web3-bridge/     # Week 3：Chain-aware Context、Tool Use、Agent Wallet、AI Security
│   └── week4-project/            # Week 4：项目构思与设计
├── handbook-feedback/            # 阅读 Handbook 时的问题和反馈
└── projects/                     # 项目设计文档和原型
```

## 学习路径

| 阶段 | 周次 | 主题 | 核心概念 |
|------|------|------|----------|
| AI 基础 | Week 1 | LLM & Prompt | LLM、Prompt、Context、Agent |
| Web3 基础 | Week 2 | 链与钱包 | Network、Cryptography、Wallet、Smart Contract |
| 交叉融合 | Week 3 | AI × Web3 | Chain-aware Context、Web3 Tool Use、Agent Wallet、AI Security |
| 项目构思 | Week 4 | 原型设计 | Agentic Commerce / Wallet Permission / AI Security / Dev Tooling |

## 笔记规范

每篇笔记文件名格式：`YYYY-MM-DD-主题.md`

笔记内容包含：
- 今日阅读的 Handbook 章节
- 核心概念理解（用自己的话写）
- 最小实践记录
- 遗留问题（同步到 handbook-feedback/）
