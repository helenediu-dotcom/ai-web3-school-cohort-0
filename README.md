# AI × Web3 School — Cohort 0 学习仓库

> Helene 的 AI × Web3 School 个人学习记录仓库

## 学员信息

- **背景**：环保行业环境咨询，AI / Web3 / 编程三方面新手
- **每日投入**：1-2 小时
- **学习周期**：8 周
- **开始时间**：2026-05

## 仓库结构

```
ai-web3-school-cohort-0/
├── notes/                      # 学习笔记（按周分组）
│   ├── week1-ai-basics/        # Week 1：LLM、Prompt 基础
│   ├── week2-ai-basics/        # Week 2：Context、Agent
│   ├── week3-web3-basics/      # Week 3：Network、Cryptography
│   ├── week4-web3-basics/      # Week 4：Wallet、Smart Contract
│   ├── week5-ai-web3-bridge/   # Week 5：Chain-aware Context、Tool Use
│   ├── week6-ai-web3-bridge/   # Week 6：Agent Wallet、AI Security
│   ├── week7-frontier/         # Week 7：前沿探索
│   └── week8-project/          # Week 8：项目构思与设计
├── practice/                   # 实践记录（代码、操作截图说明）
├── handbook-feedback/          # 阅读 Handbook 时的问题和反馈
├── project/                    # Week 7-8 项目设计文档
└── checkins/                   # 每日打卡记录
```

## 学习路径

| 阶段 | 周次 | 主题 | 核心概念 |
|------|------|------|----------|
| AI 基础 | Week 1-2 | LLM & Prompt | LLM、Prompt、Context、Agent |
| Web3 基础 | Week 3-4 | 链与钱包 | Network、Cryptography、Wallet、Smart Contract |
| 交叉融合 | Week 5-6 | AI × Web3 | Chain-aware Context、Web3 Tool Use、Agent Wallet、AI Security |
| 前沿探索 | Week 7-8 | 项目构思 | Agentic Commerce / Wallet Permission / AI Security / Dev Tooling |

## 笔记规范

每篇笔记文件名格式：`YYYY-MM-DD-主题.md`

笔记内容包含：
- 今日阅读的 Handbook 章节
- 核心概念理解（用自己的话写）
- 最小实践记录
- 遗留问题（同步到 handbook-feedback/）

## 打卡规范

打卡文件存放在 `checkins/`，格式：`YYYY-MM-DD.md`
