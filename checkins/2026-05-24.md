## 核心收获

这 7 个主题不是 7 节并列的课，而是「一条主线 + 两类支撑 + 一层横切」：Dev Stack→Network→合约→AA→DeFi 是主线；Oracle 把链外数据带进来、Indexing 把链上数据吐出去；Security 横跨全部。

在 AI×Web3 里两个最关键的交叉点：
1. **Account Abstraction 的 Session Key 是 Agent Wallet 的真正底座** — 让 AI 在「指定合约 + 指定额度 + 指定时效」内自动执行，高风险动作仍回到用户钱包确认
2. **AI Oracle 把模型推理上链** — 但「模型说什么合约就执行什么」很危险，必须配输入可追溯 + 版本记录 + 挑战/仲裁机制
