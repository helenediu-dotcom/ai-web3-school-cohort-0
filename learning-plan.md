- **姓名**：Helene

- **起点**：AI / Web3 / 编程三方面新手

## 学习策略

概念先行 + 最小实践：
- 先建立基础概念地图，不急于深入技术细节
- 每个概念配合一个最小可验证的实践
- 优先理解「为什么需要」，再学「怎么做」
- 利用 AI 工具（Claude Code）辅助理解和实践

## 学习路径

### Week 1：AI 基础概念
**目标**：理解 LLM、Prompt、Context、Agent 的基本概念

**核心内容**：
- [LLM](https://aiweb3.school/zh/handbook/ai/llm/)：大模型能做什么，不能替代什么
- [Prompt](https://aiweb3.school/zh/handbook/ai/prompt/)：如何把任务目标、边界和输出格式写清楚
- [Context](https://aiweb3.school/zh/handbook/ai/context/)：模型一次能看见什么
- [Agent](https://aiweb3.school/zh/handbook/ai/agent/)：如何让模型进入工具调用和多步执行

**最小实践**：
- 用 Claude Code 完成一个知识整理任务
- 写 3-5 个不同场景的 Prompt 并对比效果
- 记录 Prompt 改进前后的差异

### Week 2：Web3 基础概念
**目标**：理解钱包、账户、签名、合约的基本逻辑

**核心内容**：
- [Network](https://aiweb3.school/zh/handbook/web3/network/)：区块、共识、L2、RPC 和链上状态
- [Cryptography](https://aiweb3.school/zh/handbook/web3/cryptography/)：哈希、公私钥、签名
- [Wallet](https://aiweb3.school/zh/handbook/web3/wallet/)：钱包不只是登录按钮
- [Smart Contract](https://aiweb3.school/zh/handbook/web3/smart-contract/)：链上规则如何部署和调用

**最小实践**：
- 安装 MetaMask，连接测试网（Sepolia）
- 获取测试币，完成一笔转账
- 在 Etherscan 上查看交易记录

### Week 3：AI × Web3 交叉概念
**目标**：理解 AI 和 Web3 如何连接

**核心内容**：
- [Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/)：链上状态如何进入 Agent 上下文
- [Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/)：RPC、钱包、合约工具如何被 Agent 调用
- [Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/)：Agent 能拿什么权限
- [AI Security](https://aiweb3.school/zh/handbook/bridge/ai-security/)：Prompt Injection、工具滥用、权限隔离

**最小实践**：
- 用 Claude Code 读取一个链上合约的状态
- 设计一个「AI 辅助的链上数据查询」场景
- 列出这个场景的安全边界

### Week 4：前沿探索 + 项目构思
**目标**：选择一个方向，设计最小原型

**可选方向**：
- [Agentic Commerce](https://aiweb3.school/zh/handbook/tracks/agentic-commerce/)
- [Wallet / Permission](https://aiweb3.school/zh/handbook/tracks/wallet-permission/)
- [AI Security](https://aiweb3.school/zh/handbook/tracks/ai-security/)
- [Dev Tooling](https://aiweb3.school/zh/handbook/tracks/dev-tooling/)

**最小实践**：
- 写一份项目设计文档（问题、方案、技术栈、最小 MVP）
- 列出需要学习的技术点
- 准备项目展示

## 每日工作流

**学习节奏**（1-2h）：
1. 阅读 Handbook 对应章节（30-40min）
2. 完成最小实践或笔记记录（30-40min）
3. 记录问题到 `handbook-feedback/`，完成打卡草稿（20min）
4. 提交打卡到 [WCB Learning 页面](https://web3career.build/zh/programs/AI-Web3-School#tab=learning)

## 注意事项

1. **不要着急**：理解概念比完成进度更重要
2. **多问为什么**：遇到不理解的概念，先问「为什么需要它」
3. **最小实践**：每个概念都配合一个最小可验证的操作
4. **记录问题**：学习中的卡点、疑问都记录到 `handbook-feedback/`
5. **隐私安全**：本仓库是 public 的，不要放 API key、私钥、个人隐私信息
